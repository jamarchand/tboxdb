Using the T-Box Database

Searching the database
You can search TBDB using the following terms:

  • Sequence: Show T-boxes which contain your input sequence
  • Species: Show T-boxes from your input species
  • Codon: Show T-boxes which contain your input codon
  • Amino acid: Show T-boxes which recognize tRNAs for your input amino acid.
  • Note that Asp and Asn tRNAs can be recognized by the same T-boxes

Downloading the database
You can also download the database from here to use it offline.
