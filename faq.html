Using the T-Box Database

Overview of data table fields

The T-Box table contains the following visible fields:

  • T-box ID: A unique ID, generated by hashing the T-box sequence
  • Host organism: The organism in which the T-box is found
  • Accession number: The accession number of the sequence in which the T-box is present
  • Specifier: This is the predicted codon (pairs with tRNA anticodon)
  • Discriminator region: This is the UGGN sequence that pairs with the tRNA NCCA
  • Predicted amino acid/tRNA: This is the predicted amino acid / tRNA combination that binds the T-box
  • Downstream gene: This is the gene immediately downstream of the T-box. Note that some T-boxes (e.g. 0RYTHVXY) regulate polycistronic operons; in this case only the first ORF is listed.

Overview of T-box entries

Besides the characteristics visible in the main table, more T-box data can be found at the subpage for that T-box:

  • T-box sequence
  • T-box secondary structure (dot-bracket notation, as well as an image generated with VARNA)
  • Thermodynamic parameters for the predicted secondary structure

Searching the database

You can search TBDB using the search bar provided. Search terms entered will search across all fields. Multiple fields can be searched at once by including a space between search inputs.

  • Sequence: Show T-boxes which contain your input sequence. Note that sequence is a hidden field.
  • Accession: Show T-boxes found within the sequence found at the input accession number
  • Host organism: Show T-boxes from your input species
  • Codon: Show T-boxes which contain your input codon. e.g. codon=CUA
  • tRNA Family: Show T-boxes which recognize tRNAs for your input amino acid. Predicted from specifier sequence. e.g. Tyr (CUA)
  • Discriminator: Show T-boxes which contain your anti-discriminator base with adjoining anti-acceptor arm. e.g. disc=UCCA
  • Amino acid: Show T-boxes that are predicted to sense a specific amino acid. e.g. aa=TYR.
  • Found tRNA-pair: Show T-boxes that have had a tRNA pair identified by tRNAscan-SE. e.g. trna=TRUE.
  • Downstream protein: Show T-boxes with a particular class of downstream protein. e.g. transporter
