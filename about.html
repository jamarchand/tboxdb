About TBDB:

TBDB is a database of T-boxes and their predicted features.
Data sources

  • The Rfam database (12313 T-boxes, predicted using INFERNAL)
  • Vitreschak et al., 2008 (698 T-boxes, manually curated)

T-box feature prediction
The predictions contained in TBDB were performed using the method found in our Github. In summary, this method performs feature prediction in two steps. First, the query sequence is passed to INFERNAL, which uses a covariance model (RF00230.cm, obtained from Rfam) to predict secondary structure. Second, the secondary structure is searched for motifs including Stem 1, the specifier loop, and the antiterminator. The codon and discriminator are extracted from the position of these motifs.

Accuracy of results
A test set of 698 T-box sequences (from Vitreschak et al., 2008) gave the following results:

  • 694 sequences had a T-box detected by INFERNAL
  • 619 of these detected T-boxes matched the reference structure closely enough to make a prediction:
    • 555 codons were predicted correctly (90%)
    • 22 codons were off by -1 (3.5%)
    • 22 codons were off by +1 (3.5%)
    • 20 codons were otherwise incorrect (3%)

So, the predictions can be expected to be roughly 90% accurate, with a top-3 accuracy of 97%.
